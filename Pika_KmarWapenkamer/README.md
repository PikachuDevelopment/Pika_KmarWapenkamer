# Pika_KmarWapenkamer

### Wat is dit?

* Wapenkamer voor KMAR.

### Wat kan het
- Je kan wapens krijgen van een echte Kmar Medewerken.

## Download

### Met Git
```
cd resources
git clone https://github.com/qalle-fivem/qalle_policearmory
```

### Manueel
- Download dit.

## INSTALATIE
- Voeg toe aan `server.cfg`:

```
Ensure NL_Kmararmory
```
